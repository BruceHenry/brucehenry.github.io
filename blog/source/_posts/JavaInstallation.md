1. Download JDK and install it.

2. Set the environment varibles as following,

- JAVA_HOME：C:\Program Files\Java\jdk1.8.0_102
- JDK_HOME：%JAVA_HOME%
- JRE_HOME：%JAVA_HOME%\jre
- Path：......;%JAVA_HOME%\bin
- CLASSPATH：.;%JAVA_HOME%\lib;%JAVA_HOME%\jre\lib
