ArrayList and Array, Convert to Each Other

Convert ArrayList to Array in Java

List <T> list = new ArrayList<T>();
T [] array = list.toArray(new T[list.size()]);

Convert Array to ArrayList in Java

  • Flexible Length (Recommend)
List list = new ArrayList(Arrays.asList(array));
  • Inflexible
List list = Arrays.asList(array);

You are not able to add new values to it, because list object is immutable.
